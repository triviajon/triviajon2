# triviajon

a personal portfolio website built with [eleventy](https://www.11ty.dev/). showcases my work in theater, music, and software development.

## building

install dependencies and run the dev server:

```bash
npm install
npm run dev
```

## deploying

the site is deployed to github pages via a ci/cd workflow that runs on every push to `main`. the workflow builds the site and deploys it to `triviajon.com`.
